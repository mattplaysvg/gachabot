# 介紹
功能眾多，包括但不限於:  
-音樂機器人  
-抽卡系統  
-俄羅斯方塊  
-全台最大政治消費機器人，三個黨派一次滿足 (關鍵字觸發)  
## 指令查詢
ඞhelp  
## 音樂播放器 (僅支援youtube)
ඞplay <關鍵字/url> (不支援播放清單)  
ඞplaylist  
ඞrandom 打亂播放清單  
ඞincrease 把 url 收進資料庫  
ඞdecrease 刪除自己新增的 url  
ඞwhatever 隨機播放歌曲(僅抽取自己新增的)  
ඞwhatever_all  隨機播放歌曲  
部分指令被收進模塊中  
## 抽卡系統
ඞg 抽卡  
ඞsc 查詢卡片
ඞsca 根據分類查詢卡片  
ඞscs 查詢卡片持有排行  
ඞdaily 每日簽到  
ඞcard 查詢擁有的卡片  
ඞprofile 個人檔案  
ඞequip 裝備卡片  
ඞunequip 解除裝備卡片  
ඞmatch 匹配系統  
ඞc 變更顯示卡片的第一張圖片  
ඞclone 花錢印卡  
ඞtrade 交易卡片  
ඞdeleterecord 刪除自己的抽卡紀錄  
想新增其他卡片可以加入伺服器  
## 其他指令
ඞtetris 俄羅斯方塊(bata)  
ඞhello
